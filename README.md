## Human Resources
This project Explores Human Resource (HR) Analytics using data-driven Insights to help organization make smarter decisions. It focuses on analzyzing employee data to uncover trends in attrition, performance, demographics job satisfaction, and workforce diversity.


### Data Source 
The source of this data is kaggle.com The Primary dataset used in this analysis is HR.csv
